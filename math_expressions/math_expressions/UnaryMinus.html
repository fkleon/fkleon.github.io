UnaryMinus class

The unary minus negates its argument.

Extends

Constructors

new UnaryMinus(exp) #

Creates a new unary minus operation on the given expression.

For example, to create -1:

one = new Number(1);
minus_one = new UnaryMinus(one);

or just:

minus_one = new UnaryMinus(-1);
